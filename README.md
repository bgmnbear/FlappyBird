## FlappyBird
基于 canvas 实现的 2D 小游戏框架，此 Demo 实现了 FlappyBird 的核心部分。

## 框架实现功能
- 支持 Image 对象，包括基本控制(鼠标/键盘事件)
- 支持 Label 对象
- 支持 Animation 对象
- 简单的场景切换
- 简单的粒子系统
- 部分 Demo 提供关卡编辑器

## 此 Demo 实现功能
- 开始/游戏/结束场景切换
- 碰撞检测
- Bird 对象支持重力以及旋转
- Pipe 对象随机高度
- 记分牌
